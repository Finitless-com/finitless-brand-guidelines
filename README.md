# Finitless Brand Guidelines

Official brand assets and usage guidelines for Finitless.

## Structure

```
brand-guidelines/
├── logos/
│   ├── primary/          # Main logo (full color)
│   │   ├── svg/
│   │   └── png/
│   ├── secondary/        # Alternate versions (white, dark, monochrome)
│   │   ├── svg/
│   │   └── png/
│   ├── icon/             # Favicon, app icon, social avatar
│   │   ├── svg/
│   │   └── png/
│   └── wordmark/         # Text-only version
│       ├── svg/
│       └── png/
├── typography/
│   ├── fonts/            # Font files (.woff2, .ttf, .otf)
│   └── TYPOGRAPHY.md     # Font families, sizes, weights
├── colors/
│   └── COLORS.md         # Palette with hex, RGB, HSL values
├── imagery/
│   ├── photography/      # Brand photo style references
│   ├── illustrations/    # Custom illustrations
│   └── patterns/         # Brand patterns and textures
├── templates/
│   ├── social-media/     # Social media templates
│   ├── presentations/    # Slide templates
│   └── email/            # Email templates
├── icons/                # Custom icon set
├── usage-examples/       # Real-world usage examples
└── BRAND-GUIDELINES.md   # Complete brand guide
```

## Quick Reference

| Asset | Location |
|-------|----------|
| Main logo (SVG) | `logos/primary/svg/` |
| Logo for dark backgrounds | `logos/secondary/svg/` |
| Favicon / App icon | `logos/icon/` |
| Brand fonts | `typography/fonts/` |
| Color palette | `colors/COLORS.md` |
| Full guidelines | `BRAND-GUIDELINES.md` |
