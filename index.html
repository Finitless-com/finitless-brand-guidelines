Brand Guidelines

The single source of truth for the Finitless brand identity. For humans and AI agents.

Logo

The Finitless mark combines the letter N with the infinity symbol, representing limitless possibilities. The wordmark uses the BEASIGNE typeface.

Variants

Color on dark (preferred) — finitless-logo-horizontal-on-dark.png
Color on light — finitless-logo-horizontal-on-light.png

Brand Stamp (Icon)

On dark background — finitless-icon-on-dark-2000.png
On light background — finitless-icon-on-light-2000.png

Rules

  • Never stretch the logo — always resize proportionately
  • Never modify or recreate the icon — use supplied files only
  • Minimum clear space: 50% of icon height on all sides
  • Minimum logo height: 24px (icon), 80px width (full logo)

Typography

Logo font: BEASIGNE (Regular) — used exclusively for the wordmark.

UI font: Inter — weights 400, 600, 700, 800 — used for all text.

Type Scale

  • display Never Miss 72px / 800
  • h1 Brand Guidelines 48px / 800
  • h2 Section Heading 36px / 700
  • h3 Sub-Section 24px / 700
  • body-lg AI-powered voice and WhatsApp agents. 18px / 400
  • body Default body text for paragraphs and descriptions. 16px / 400
  • caption Labels and fine print 12px / 400
